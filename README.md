# 📘 ESTUDO_PHP

Repositório criado para estudos e prática da linguagem **PHP**, abordando desde conceitos básicos até estruturas mais avançadas da linguagem.

O objetivo deste projeto é consolidar conhecimentos fundamentais em PHP através de exemplos práticos e organizados por tema.

---

## 📂 Estrutura do Projeto

O repositório está organizado por arquivos separados de acordo com o conteúdo estudado:

### 🔹 Fundamentos
- `index.php`
- `variaveis_constantes.php`
- `variaveis_PHP.php`
- `casting_tipos.php`
- `false_null_empty.php`
- `echo_print.php`
- `tag.php`

### 🔹 Operadores
- `operadores_aritmeticos.php`
- `operadores_incremento_decremento.php`
- `operador_ternario.php`

### 🔹 Estruturas Condicionais
- `ifelse_operadores_condicionais.php`
- `switch.php`

### 🔹 Estruturas de Repetição
- `lacos_rep.php`
- `loops_pratica2.php`

### 🔹 Arrays
- `array_basico.php`
- `array_multidimensional.php`
- `array_pesquisa.php`

### 🔹 Funções
- `functions.php`

- `functions_math.php`

## 🎯 Conteúdos Abordados

- Sintaxe básica do PHP
- Tipos de variáveis
- Operadores aritméticos e lógicos
- Estruturas condicionais (`if`, `else`, `switch`)
- Estruturas de repetição (`for`, `while`, `foreach`)
- Manipulação de arrays
- Funções nativas do PHP
- Criação de funções personalizadas
- Manipulação de datas, strings e números
